# 诗歌集

作者：玄兴梦影（笔名：沐雪程序员）

## 简介

这是一部个人诗集，收录了作者从学生时代至今创作的诗歌作品，按主题分为七个部分：

- 【爱情诗篇】19首
- 【人生感悟】17首
- 【自然写景】10首
- 【亲情友情】8首
- 【校园青春】3首
- 【独处心语】4首
- 【其他诗作】3首

## 在线阅读

本诗集已发布为 GitHub Pages 网站，欢迎在线浏览：

**[https://mxuexxmy.github.io/poems-collection/](https://mxuexxmy.github.io/poems-collection/)**

网站采用中国风设计，以宣纸色为底，朱砂红与淡竹青为点缀，营造优雅的阅读体验。

## 本地预览

克隆本仓库后，直接在浏览器中打开 `index.html` 文件即可预览：

```bash
git clone https://github.com/mxuexxmy/poems-collection.git
cd poems-collection
# 然后用浏览器打开 index.html
```

或使用本地服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve

# 然后访问 http://localhost:8000
```

## 内容说明

- `index.html` - 主页面（中国风样式，包含所有诗歌）
- `styles.css` - 中国风样式表
- `.nojekyll` - GitHub Pages 配置文件
- `诗歌集.md` - 原始 Markdown 格式诗歌集
- `诗歌.md` - 原始诗歌文件
